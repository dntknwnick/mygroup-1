🔐 Role-Based Authentication System

React Application with Three User Roles

📋 Project Overview

This is a complete React application with role-based authentication supporting three user types:

  • Super Admin - Can create Corporate users
  • Corporate - Can create Branch users
  • Branch - Access to branch-specific dashboard

🚀 Setup Instructions

Prerequisites: Node.js and npm must be installed on your system.

Step 1: Install Node.js

Download and install Node.js from nodejs.org

Step 2: Install Dependencies

cd role-based-auth
npm install

Step 3: Start Development Server

npm run dev

Step 4: Open Application

Navigate to http://localhost:3000 in your browser

✨ Key Features

  • 🔐 Role-based authentication and authorization
  • 🛡️ Protected routes with automatic redirection
  • 📱 Responsive design with React-Bootstrap
  • 👥 User management (create subordinate users)
  • 📊 Role-specific dashboards
  • 🎨 Modern UI with gradient backgrounds
  • 💾 Local storage for session persistence

🔧 Technologies Used

  • React 18 - Frontend framework
  • React Router DOM - Client-side routing
  • React-Bootstrap - UI components
  • Bootstrap 5 - CSS framework
  • Vite - Build tool and dev server

🎯 Testing the Application
